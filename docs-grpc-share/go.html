WHY GO !!!

go 的优势

  • 开源的，大公司做背书
  • 编译快，静态编译，极易部署
  • 跨平台
  • 语法简单
  • 天然的并发

Go 的基本使用

package main

import (
    "fmt"
    "time"
)

func main() {
    go fmt.Println("go")
    time.sleep(time.Second)

    fmt.Println("hello world")
}

$ go build hello-world.go
$ ls
hello-world hello-world.go

$ ./hello-world
go
hello world
